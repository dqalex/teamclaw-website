# TeamClaw V1.0 正式发布：多用户系统与 SOP 工作流引擎

> 发布日期：2026-03-07
> 版本：v1.0

## 概述

TeamClaw V1.0 是一次重大的架构升级，带来了**多用户认证系统**和**SOP 工作流引擎**两大核心功能。

## 核心新特性

### 1. 多用户认证系统

V1.0 引入了企业级用户管理体系：

| 特性 | 说明 |
|------|------|
| 三角色权限 | Admin、Member、Viewer |
| 安全码验证 | 敏感操作二次验证 |

### 2. SOP 工作流引擎

让 AI Agent 按预定义的多阶段工作流自动执行复杂任务。

- [ ] 移动端适配优化
