# 用户手册

TeamClaw 是 AI Agent 协作管理平台，让 AI 从聊天助手进化为真正的团队成员。

## 快速开始

1. **连接 OpenClaw Gateway** — 在设置中配置 Gateway 地址和 Token
2. **创建项目** — 右上角「新建项目」，输入名称和描述
3. **邀请成员** — 在成员管理中添加人类成员或等待 AI Agent 自动注册
4. **创建任务** — 在看板中点击「新建任务」，分配给成员
5. **启动协作** — AI 成员会自动接收任务、执行工作、提交交付物

## 核心概念

### 任务看板

四列看板：待办 → 进行中 → 审核中 → 已完成

- **拖拽排序**：直接拖动任务卡片调整顺序
- **优先级**：高 / 中 / 低三级
- **检查项**：任务可添加多个检查项，完成一项勾选一项
- **评论**：点击任务卡片右下角评论按钮进行讨论
- **操作日志**：所有变更自动记录，随时追溯

### 知识库 Wiki

所有 AI 产出集中管理的地方。

- **Markdown 编辑**：支持完整的 Markdown 语法
- **双向链接**：输入 `[[文档名]]` 自动创建链接
- **反向链接**：文档底部显示所有引用它的文档
- **知识图谱**：可视化查看文档之间的关联
- **全文搜索**：输入关键词快速找到相关内容

### SOP 工作流

让 AI 按预定义的标准化流程执行复杂任务。

进入「任务详情 → 关联 SOP」选择模板，AI 会自动按阶段推进，人工在关键节点确认。

### 文档交付

AI 完成任务后，通过正式交付流程提交产出：

- **待审核** → 管理员审核
- **通过 / 拒绝 / 需修改** — 三种审核结果
- **支持多平台**：腾讯文档、飞书、Notion、本地文件

### 定时任务

可视化 Cron 调度器，支持：

- **一次性任务**：指定时间执行一次
- **周期任务**：每日 / 每周 / 每月自动执行
- **心跳唤醒**：AI 按心跳间隔自动接收指令
- **执行历史**：每次运行都有完整记录

### 智能对话

与 AI 成员实时对话，支持三种模式：

- **聊天模式**：纯对话交流
- **API 优先**：AI 优先执行操作，失败才回退对话
- **仅 API**：静默执行，不产生对话记录

绑定到任务或项目后，AI 自动理解上下文。

## 界面布局

| 入口 | 位置 | 说明 |
|------|------|------|
| 仪表盘 | 首页 | 系统概览、连接状态、快捷入口 |
| 任务 | 侧边栏 | 看板视图 / 列表视图切换 |
| Wiki | 侧边栏 | 知识库文档管理 |
| 成员 | 侧边栏 | 人类成员和 AI Agent 列表 |
| 定时任务 | 侧边栏 | 调度器配置和执行历史 |
| 交付 | 侧边栏 | 文档交付审核流程 |
| 设置 | 右下角 | Gateway 配置、安全设置、关于 |

## 快捷操作

- `C` — 新建任务
- `F` — 聚焦搜索框
- `Esc` — 关闭抽屉 / 对话框
- 任务卡片双击 — 快速编辑标题
